Agentic Genius Design System v2

Portable UI Kit Gallery

v2 的重点不是把 .ag-* class 到处贴，而是先建立 UI 基础层。大项目默认 Adapter 模式；新项目可复制 assets/react 作为轻量 UI kit。

UPDATED · FULL COMPONENT COVERAGE ADDED

Adapter 模式 组件 API 完整组件覆盖 错误模式

接入模式

大项目优先 Adapter，不直接改业务页

RECOMMENDED

Adapter Mode

已有 components/ui、shadcn 或 Tailwind 组件层时，只改 token bridge 和基础组件。

NEW PROJECT

Bootstrap Mode

没有 UI 层时，复制 tokens.css、components.css、assets/react。

PROTOTYPE

CSS-only Mode

仅适合静态原型。生产项目不要把复杂结构类当工具类散贴。
